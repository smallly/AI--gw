# AI赛博旅伴 官网

一个基于React + TypeScript + Vite的现代化官网项目，展示AI赛博旅伴的产品功能和服务。

## 📸 项目预览

![首图预览](./public/首图.png)

## ✨ 项目特色

- 🚀 现代化技术栈：React 18 + TypeScript + Vite
- 🎨 优雅界面设计：TailwindCSS + 自定义动画
- 📱 响应式布局：完美适配各种设备
- ⚡ 快速构建：Vite 热更新开发体验
- 🎭 丰富交互：流畅的动画效果

## 🛠️ 技术栈

- **前端框架**: React 18
- **类型系统**: TypeScript
- **构建工具**: Vite
- **样式方案**: TailwindCSS + PostCSS
- **图标库**: Lucide React
- **代码规范**: ESLint

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
npm install
```

### 开发调试

```bash
npm run dev
```

访问 http://localhost:5173 查看项目

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 🎯 功能特性

### 核心功能
- **智能收集**: 随手记录生活点滴
- **AI整理**: 智能分析和归类
- **地图呈现**: 可视化展示旅行轨迹

### 界面特色
- **渐进式动画**: 优雅的页面加载效果
- **响应式设计**: 适配桌面端和移动端
- **现代化UI**: 简洁美观的用户界面

## 🔧 开发指南

- **Vite配置**: `vite.config.ts`
